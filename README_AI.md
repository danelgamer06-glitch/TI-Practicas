# Chatbot sencillo (ai_simple.py)

Archivo `ai_simple.py`: un chatbot simple basado en reglas y con una función para "enseñar" nuevas respuestas.

Uso rápido:

```bash
python ai_simple.py
```

Comandos útiles:
- Escribe mensajes normalmente. Responde a saludos y despedidas.
- Para enseñar una nueva respuesta usa:
  `enseñar pregunta | respuesta`
- Escribe `salir` para cerrar.

El conocimiento se guarda en `ai_kb.json` en el mismo directorio.
